# cwaf rules backup

* Git clone scripts

  ```bash
  git clone https://github.com/charlietag/cwaf_charlietag.git
  ```

* Setup `conf/cwaf_param.cfg`

  ```bash
  PARAM_COMODO_RULES_account="xxx@gmail.com"
  PARAM_COMODO_RULES_password="xxxxxxxxxxxxx"
  ```

* Start to download latest cwaf rules

  ```bash
  ./start.sh
  ```
